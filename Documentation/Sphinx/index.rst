Overview
--------

Exo Mass Checker is an advanced automation tool that provides:

* **Bulk Account Verification**: Efficiently verify multiple Epic Games accounts
* **Browser Automation**: Advanced browser automation with stealth capabilities
* **Cloudflare Bypass**: Comprehensive Cloudflare challenge solving
* **Telegram Integration**: User-friendly Telegram bot interface
* **Resource Management**: Intelligent resource monitoring and cleanup
* **Cloud Storage**: Automatic result storage and sharing

Architecture
------------

The system is built with a modular, asynchronous architecture:

* **Core Application**: Main bot orchestration and system initialization
* **Telegram Handlers**: Message and callback query processing
* **Account Verification**: Core business logic for account checking
* **Browser Management**: Advanced browser automation and resource management
* **Cloudflare Solvers**: Multiple solver implementations for challenge bypass
* **Utilities**: Supporting modules for file management, monitoring, and integration

Key Features
------------

Browser Automation
~~~~~~~~~~~~~~~~~~

* **Multi-Browser Support**: Camoufox, Patchright, and Playwright integration
* **Stealth Capabilities**: Advanced fingerprinting protection
* **Proxy Support**: Full proxy rotation with authentication
* **Resource Management**: Intelligent cleanup and memory management

Cloudflare Bypass
~~~~~~~~~~~~~~~~~

* **Turnstile Solver**: Dedicated Turnstile challenge solving
* **Multiple Solvers**: Fallback mechanisms for reliability
* **API Integration**: RESTful API for challenge solving
* **Challenge Detection**: Automatic challenge type identification

Account Verification
~~~~~~~~~~~~~~~~~~~

* **Epic Games Integration**: Direct Epic Games API integration
* **Authentication Flow**: Complete OAuth2 authentication
* **Status Detection**: Comprehensive account state analysis
* **Result Categorization**: Organized result classification

Telegram Bot Interface
~~~~~~~~~~~~~~~~~~~~~

* **Interactive Keyboards**: User-friendly inline keyboards
* **File Upload**: Support for account and proxy file uploads
* **Progress Monitoring**: Real-time checking progress updates
* **Result Management**: Categorized result downloads
